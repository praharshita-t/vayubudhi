# Member 4: LangGraph Advisory Agent

This directory contains the LangGraph AI multi-agent coordinator that leverages Google Gemini APIs to formulate citizen warnings and advisory notifications.

## Key Tasks & Roles
1. **LangGraph Coordinator**: Routes flows between source prediction outputs, VRP optimization paths, and final multilingual summaries.
2. **Gemini Advisory Generation**: Designs templates and invokes Gemini LLM prompts to construct citizen alerts in English, Hindi, and Kannada.

## Checkpoints
- **Days 4–6**: Construct prompts and agent nodes using local mock attribution and forecast JSON payloads.
- **Day 7**: Connect LangGraph node interfaces directly to real backend API models.
